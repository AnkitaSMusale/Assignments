1). Navigate to the top-level of the project or the specific repository you want to download.
2). Click the green "Code" download button on the top-level page.
3). Select the "Download ZIP" option from the "Code" pull-down menu.
4). After downloading zip file. Extract the file on your device and open it on visual studio
